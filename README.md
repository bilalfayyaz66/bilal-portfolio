# Bilal Fayyaz Portfolio

Simple, professional portfolio built with React + Vite and designed for Vercel deployment.

## Edit portfolio content

Most content is in:

```txt
src/data.js
```

Edit your intro, skills, contacts, and project text there.

## Run locally in VS Code

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal, usually:

```txt
http://localhost:5173
```

## Build for production

```bash
npm run build
```

## Deploy on Vercel

1. Push this folder to a GitHub repository.
2. Open Vercel.
3. Click Add New Project.
4. Import your GitHub repository.
5. Keep defaults:
   - Framework Preset: Vite
   - Build Command: npm run build
   - Output Directory: dist
6. Click Deploy.

## Notes

- Social links are shown as icons only.
- Styling is intentionally simple: white background, black text, borders, no heavy colors, no shadows.
